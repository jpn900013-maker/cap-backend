Developer Guide

Integrate 9Captcha directly into your automation pipelines. 100% headless API process.

1. Create a Solve Task

Send the challenge parameters to our backend to enqueue an AI job.

POST /create_task
  • key: Your API Key
  • type: "hcaptcha_basic"
  • data.sitekey: Captcha widget sitekey
  • data.siteurl: Target domain (e.g. "discord.com")
  • data.rqdata: Request data string
import requests

SERVER_URL = "http://localhost:5000"
API_KEY = "YOUR_API_KEY_HERE"

payload = {
    "key": API_KEY,
    "type": "hcaptcha_basic",
    "data": {
        "sitekey": "4c67...", 
        "siteurl": "discord.com",
        "rqdata": "YOUR_RQDATA_STRING"
    }
}

resp = requests.post(f"{SERVER_URL}/create_task", json=payload)
print(resp.json()) # Output: {"status": "success", "task_id": "..."}

2. Poll for Solution

Request the result every few seconds until the job completes.

GET /get_result/<task_id> 
import time

task_id = "..." # From previous step
timeout = 120
start = time.time()

while time.time() - start < timeout:
    time.sleep(3)
    resp = requests.get(f"{SERVER_URL}/get_result/{task_id}", json={"key": API_KEY})
    data = resp.json()
    
    if data["status"] == "success":
        solution = data["solution"]
        print("Solved!", solution[:20])
        break
    elif data["status"] == "error":
        print("Error:", data["message"])
        break
